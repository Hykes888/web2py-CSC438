Purpose of the web application

 

heicheleList purpose is to allow people in need after a declared natural disaster. The website will provide a way for the people to search

for products they need but don’t have the means to afford the product. The products are donated from user that have the means to donate

a product or have the means to donate money for delivery of a product. So the vision of the webpage is when in need: Find your need.

 

Workflow of heicheleList

 

Home page

 

 

The home page allows you to sign up for heicheleList, and login, both sign up and login pages will take you the home navigation page of heicheleList.

 

Login and sign up page

 

 

 

 

 

home navigation page

 

 

Provides three options for the main purpose of the web page. Donate a product, Product Search, and Donation for delivery.

And once the user is logged in the navbar provides functionality to always come back to the home navigation page, update profile, My HeicheleList,

change password, and logout.

 

Purpose 1: Donate a product

 

Product Entry page

 

 

Allows the user to enter information about the product they are about to donate.

 

Product home address page

 

 

 

It is required for the product that is being donated to have a source address to be able estimate the shipping information and to actually ship the product.

 

Product parcel information page

 

 

 

This is also required to be entered because we need the parcel size to be able to estimate the shipping information. Currently we are only using USPS

for shipping. Once all the information is entered for donation it will bring you back to the home navigation page.

 

Purpose 2: Product Search

 

Product Search page

 

 

This page provides all the products that has been donated. User can click on the image to reveal more details about the product. And search for a product as well.

 

Product details page

 

 

Allows user to see and read more about the details of the product. And if the user needs this product he can either go back to product search or request the product for shipment.

 

Detestation delivery information page

 

 

Allows the user to input where the product needs to be delivered has to be a valid US address.

 

Confirmation page

 

Notifies the user that the product is ready for a donor to pay for the delivery of the product. And at that point no one can also request delivery of that product.

 

Purpose 3: Donate for Delivery

 

Delivery donation search page

 

Allow the user to search for the donation estimated delivery cost that will fit his budge.  Click on the image for the product for which the user would like to donate

the user money for delivery charges.

 

Delivery choice page

 

 

Allows the user to choose which shipping rate and estimated time of delivery he would like to donate.

 

Donation payment

 

 

User can enter credit card information and submit payment.

 

Confirmation Page

Notifies the user that the user donation has been charged and shipping of the product is in process.
